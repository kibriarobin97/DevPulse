# DevPulse API

A collaborative platform for software developer teams to report bugs, suggest features.

## Live URL
```
https://dev-pulse-roan-xi.vercel.app
```

## Features
- User registration and login with JWT authentication
- Role-based access control (contributor & maintainer)
- Create, read, update, and delete issues
- Filter issues by type and status
- Sort issues by newest or oldest

## Tech Stack
- Node.js
- TypeScript
- Express.js
- PostgreSQL (NeonDB)
- bcryptjs
- jsonwebtoken
